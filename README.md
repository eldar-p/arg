# Каталог Манделы — Локальный архив

Интерактивный аналоговый хоррор в браузере по мотивам ARG **The Mandela Catalogue**.

Неофициальный трибьют. Не связан с Alex Kister и оригинальным сериалом.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080

## Что внутри

- CRT/VHS-атмосфера, сканлайны, помехи
- Три сюжетные ветки: оператор горячей линии, один дома, архивариус
- Звонки, учебные плёнки APS, сверка лиц (найди Альтерната)
- Паранойя, очки выживания и несколько концовок
- Звук через Web Audio API (гул, звонок, stinger)

## Управление

Только мышь / тап. Наушники желательны. Есть вспышки и резкий звук.
