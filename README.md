# Каталог Манделы — некоммерческая фанатская медиа-игра

Бесплатная интерактивная медиа-игра / VHS-архив по мотивам **The Mandela Catalogue**.

## Disclaimer

**Это некоммерческий фанатский проект. Все права на франшизу, персонажей и лор принадлежат Алексу Кистеру (Alex Kister). Проект не связан с официальными создателями.**

Тот же текст показывается при первом запуске игры и на главном экране.

## Правила фанатского проекта

- Полностью **бесплатно**: нет продажи, микротранзакций, платного доступа и коммерческой рекламы.
- Донаты (если появятся) — только добровольно, на покрытие хостинга.
- Название оригинала и автор указаны на сайте и в игре.
- Используются собственные/сток-ассеты трибьюта (Unsplash, Mixkit) + оригинальный код проекта.

## Медиа

### Официальные кассеты
Эпизоды **Alex Kister** подключаются через **YouTube embed** (кнопка «ОФИЦИАЛЬНЫЕ КАССЕТЫ»).  
Файлы сериала **не скачиваются и не хранятся** в репозитории — так соблюдается копирайт при фанатском использовании.

### Фан-адаптация
- Видео / музыка / SFX — Mixkit (`assets/video`, `assets/music`, `assets/sfx`)  
- Портреты — **фан-арт** analog-horror (`assets/portraits/*_photo.jpg`), не сток-фото  
- Атрибуции: `assets/*/ATTRIBUTION.txt`
