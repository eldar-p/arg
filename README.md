# Каталог Манделы — полный сюжет (трибьют)

Интерактивный аналоговый хоррор в браузере: **пересказ полного сюжета** ARG / сериала **The Mandela Catalogue**.

Неофициальный трибьют. Не связан с Alex Kister. События пересказаны своими словами.

## Запуск

```bash
python3 -m http.server 8080
```

Откройте http://localhost:8080 или файл `index.html`.

## Сюжет по эпизодам

1. **Ep.0 — Overcast** — ложный «Гавриил», *Think of someone you love*, начало подмены веры  
2. **Ep.1 — Vol.1** — Марк Хитклифф, звонок «Сезара», дверь, Альтернат, Тэтчер Дэвис, плёнка APS  
3. **Ep.2 — Vol.2** — Адам Мюррей и Джона Маршалл (Bythorne), дом, камера, разделение  
4. **Ep.3 — Падение Манделы** — Тэтчер, Сара, вопрос об Адаме, пустой округ, финальная сверка  

Можно пройти **с начала** или выбрать эпизод в меню.

## Персонажи

Марк, Сезар, Тэтчер, Рут, Сара, Адам, Джона, Дейв, «Гавриил», Альтернат — досье в каталоге лиц.

## Звук

Оригинальная библиотека Web Audio (хор, шёпот, стук, телефон, EAS и т.д.).  
**Не** аудиодорожки из сериала (авторские права).

## Предупреждение

Вспышки, резкий звук, психологический хоррор.
